От ветра сильного деревья
Своими кронами шумят,
Под шум закуталась деревня,
Дома под шумом тихо спят.

Шумят деревья монотонно,
Как колыбельную поют -
Послушай их - определенно,
Они симфонию ведут.
```
	[* АвтВариантИзКнижки5- Ведут симфонию свою.]
```

Шумят, как примы, все берёзы,
Лаская утомленный слух
И навевают шумом грёзы,
Смиряя беспокойный дух.

```

Успокоительно хорош
И вот уж ты рекой безбрежной
```
	[* АвтВариантИзКнижки5- Как будто ты рекой безбрежной]
```
Куда-то медленно плывешь.

Шумят надменно и спокойно
```
	[* АвтВариантИзКнижки5- Послушай, шепчутся спокойно]
```
Седые ели и сосна,
Из-за шептанья этих хвойных
Не отряхнуться ото сна.

Их мерный мах к колыханье
```
	[АвтИспр- Размерный мах к колыханье]
```
```
	[* АвтВариантИзКнижки5- Вершин сосновых колыханье]
```
Внушают благость и покой
И замедляется дыханье
Под их магической рукой.

А ветер в черном полуфраке
```
	[* АвтВариантИзКнижки5- А сильный ветер в черном фраке]
```
Все дирижирует своё
И ты один в шуршащем мраке,
```
	[АвтИспр- И я один в шуршащем мраке,]
```
Один идёшь в небытиё...
```
	[АвтИспр- Один иду в небытие...]

Во всё сгущающейся сини
И от карающих десниц...
```
	[* АвтВариантИзКнижки5- Не вижу я твоих десниц.]
```
Ты спишь под лепетом осины,
Закрывшись бархатом ресниц.
